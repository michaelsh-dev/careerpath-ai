# 🚀 CareerPath AI

CareerPath AI is a web application that helps users generate a personalized career strategy based on their current skills, career interests, and experience level.

It is designed for students, fresh graduates, and beginner developers who need clear direction on what to learn, what to build, and how to move toward their target career.

---

## ✨ Features

- Input current skills, career goals, and experience level  
- AI-generated personalized career roadmap  
- Skill gap analysis  
- Recommended skills to learn  
- Project ideas for portfolio building  
- Actionable next steps  

---

## 🧠 AI Integration

CareerPath AI uses OpenRouter API (GPT-4o-mini) as its core engine.

User input is sent to a Node.js backend, where it is processed into a structured prompt. The AI model analyzes the data and generates a clear, actionable career strategy.

This is not a simple chatbot — the AI acts as a **career decision engine**, transforming user input into meaningful guidance.

---

## 🛠 Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js + Express  
- **AI API:** OpenRouter (GPT-4o-mini)  
- **Deployment:** Vercel (frontend) + Render (backend)
